# Chalk

Chalk Academic environment

Build Django:

```
cd backend
pip install virtualenv
virtualenv env
**Windows:** source env/Scripts/activate
**Mac:** source env/bin/activate
pip install -r ./requirements.txt
python manage.py migrate
```

Run Django:

```
cd backend
**Windows:** source env/Scripts/activate
**Mac:** source env/bin/activate
python manage.py runserver
```

Build/Run React:

```
cd frontend
npm start
```
